# AuthSystem (Node.js + Express + JWT)

A simple authentication system built with **Node.js**, **Express**, **JWT**, and **bcrypt**.  
Supports **Signup**, **Login**, **Logout**, and **Protected Routes** using HTTP-only cookies.
